# Looking For Group (LFG) Mobile APP

## Start the project

### install dependencies

yarn

```bash
yarn install
```

then

```bash
yarn expo install --fix
```

npx

```bash
npm install
```

then

```bash
npx expo install --fix
```

### run project

yarn

```bash
yarn expo start
```

npx

```bash
npx expo start
```

## Tech Stack

- React-Native expo
- Typescript
- Firebase
- Redux toolkit
